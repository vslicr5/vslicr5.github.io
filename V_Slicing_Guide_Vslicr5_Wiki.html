Introduction

V-Slice is the modern charting format used in Friday Night Funkin' Polymod for creating custom charts and note patterns. It provides enhanced flexibility and features compared to legacy charting formats.

Note: This guide assumes you have basic knowledge of FNF charting and the Polymod chart editor interface.

What is V-Slice Charting?

V-Slice (Vertical Slice) is a charting system that allows for precise note placement and advanced gameplay mechanics. Unlike legacy formats, V-Slice supports custom note types, events, and modular song structures.

Key Features

  • Precision: Sub-step note placement for complex patterns
  • Custom Note Types: Support for mines, warning notes, and special mechanics
  • Event System: Built-in camera movement and stage events
  • Compatibility: Works with all FNF Polymod engines

Getting Started

Chart Editor Setup

To begin charting with V-Slice, you'll need to set up your environment properly:

  1. Open the FNF Polymod Chart Editor (Press 7 in-game)
  2. Load your song audio files (Inst and Voices)
  3. Set the BPM and time signature
  4. Configure difficulty settings
{
  "version": "2.0.0",
  "scrollSpeed": {
    "easy": 1.5,
    "normal": 2.0,
    "hard": 2.5
  },
  "playData": {
    "characters": {
      "player": "bf",
      "opponent": "dad",
      "girlfriend": "gf"
    }
  }
}
Warning: Make sure your audio files are synced properly before placing notes. Use the metronome to verify BPM accuracy.

Basic V-Slice Notation

Here's how to create basic note patterns in V-Slice format:

// Standard 4-note pattern
{
  "t": 0,      // Time in milliseconds
  "d": 0,      // Direction (0=Left, 1=Down, 2=Up, 3=Right)
  "l": 0,      // Length (0 for normal notes, >0 for holds)
  "p": "normal" // Note kind/type
}

Advanced Charting Techniques

Once you've mastered the basics, you can explore more advanced V-Slice charting techniques to create unique gameplay experiences.

Custom Note Types

V-Slice supports various custom note types for different gameplay mechanics:

"The flexibility of V-Slice note types allows charters to create unique gameplay experiences that weren't possible in legacy formats." - KadeDev

Available Note Types

// Mine note (damages player when hit)
{
  "t": 1000,
  "d": 1,
  "p": "mine"
}

// Warning note (must NOT be hit)
{
  "t": 2000,
  "d": 2,
  "p": "warning"
}

// Hold note (sustain)
{
  "t": 3000,
  "d": 3,
  "l": 500,  // 500ms hold length
  "p": "normal"
}
Tip: Use warning notes sparingly to create tension in your charts. Too many can overwhelm players and make patterns feel unfair.

Camera Events and Stage Triggers

V-Slice includes a robust event system for controlling camera movement and stage effects:

// Camera focus event
{
  "t": 4000,
  "e": "focusCamera",
  "v": {
    "char": 1,  // 0=Player, 1=Opponent, 2=Girlfriend
    "ease": "linear",
    "duration": 0.5
  }
}

// Stage light event
{
  "t": 4500,
  "e": "setStageLight",
  "v": {
    "color": "#FF0000",
    "intensity": 1.5
  }
}
Event Timing Best Practices

When implementing events in your charts, keep these guidelines in mind:

  • Sync camera movements to the beat or significant musical moments
  • Avoid rapid camera switches that can cause motion sickness
  • Test your events on different screen sizes
    • 16:9 aspect ratio (most common)
    • 4:3 aspect ratio (older monitors)
    • 21:9 ultrawide displays
  • Document complex event sequences for future reference
