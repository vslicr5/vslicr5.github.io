VslicR5 - FNF Chart Editor

VslicR5 is a web-based chart editor for Friday Night Funkin' V-Slice/Polymod format. Created by YZY MOB, this tool allows you to create, edit, and export custom charts directly in your browser with real-time audio visualization and an intuitive interface.

Current Version: v0.1 - Active development with v1.1 coming soon!

Overview

VslicR5 is designed specifically for charting Friday Night Funkin' songs in the V-Slice/Polymod format. Whether you're creating charts for custom mods or official content, VslicR5 provides all the tools you need in a single, accessible web application.

Why VslicR5?

  • No Installation Required: Run entirely in your browser
  • Real-time Preview: See and hear your chart as you create it
  • Multi-Difficulty Support: Chart Easy, Normal, and Hard simultaneously
  • Auto-Save: Your work is saved automatically with IndexedDB

Features

Core Charting Features

Visual Chart Editor

The main canvas displays your chart with 8 lanes (4 for player, 4 for opponent) with color-coded arrows:

Lane Side Direction Color
0-3 Player Left, Down, Up, Right Purple, Cyan, Green, Red
4-7 Opponent Left, Down, Up, Right Purple, Cyan, Green, Red (Darker)

Note Types

  • Standard Notes: Basic hit notes placed at specific timestamps
  • Hold Notes: Sustained notes with adjustable length
  • Smart Snapping: Notes automatically snap to the nearest lane for precise placement

Multi-Difficulty Charting

VslicR5 includes a unique difficulty system that allows you to chart multiple difficulties at once:

  • Toggle Easy, Normal, and Hard difficulties independently
  • Add notes to selected difficulties simultaneously
  • Filter JSON output by active difficulties
  • Visual indicators show which difficulties are active
Tip: Use the checkbox buttons to toggle difficulties on/off. Only notes for active difficulties will be added when you place notes.

Audio Playback

Import and play audio files in multiple formats:

  • OGG (recommended for FNF)

Interface Guide

Canvas Controls

The main canvas shows your chart with several visual elements:

  • Lane Dividers: Gray vertical lines separating the 8 lanes
  • Hit Line: Red horizontal line in the center (current playback position)
  • Notes: Colored arrows positioned based on time and lane
  • Hold Trails: Semi-transparent bars showing hold note duration
  • Selection Highlight: Yellow outline around selected notes

Note Placement

Using the Note Pads

Two button grids allow you to place notes:

Pad Keys Lanes Color Theme
Opponent W, A, S, D 4-7 Red background
Player Arrow Keys 0-3 Blue background

Audio Controls

- Start audio playback and chart animation
- Pause at current position
- Return to beginning (0ms)

Time Adjustment

  • -100ms Button: Jump backward 100 milliseconds
  • +100ms Button: Jump forward 100 milliseconds
  • Time Scrollbar: Drag to any point in the song

Keyboard Controls

VslicR5 supports comprehensive keyboard shortcuts:

Key Action
Arrow Left Place player left note
Arrow Down Place player down note
Arrow Up Place player up note
Arrow Right Place player right note
A Place opponent left note
S Place opponent down note
W Place opponent up note
D Place opponent right note
Space Toggle play/pause
M Toggle mobile view (experimental)
Note: Keyboard shortcuts are disabled when the JSON textarea is focused.

Workflow Guide

Creating Your First Chart

  1. Import Audio: Click the music icon and select your song file (OGG recommended)
  2. Select Difficulties: Toggle which difficulties you want to chart (Easy/Normal/Hard)
  3. Place Notes: Use keyboard shortcuts or button pads to add notes at the current time
  4. Adjust Timing: Drag notes vertically to fine-tune their timing
  5. Add Holds: Click a note and edit its length value
  6. Preview: Press Space or click Play to test your chart
  7. Save: Click the save icon to store your chart locally
  8. Export: Click the export icon to download your chart as JSON

JSON Live Editing

The JSON textarea updates in real-time and accepts manual edits:

  • Edit note properties directly in JSON
  • Changes apply immediately to the canvas
  • Supports both object and array note formats
  • Auto-converts between formats as needed

Difficulty Filtering

The JSON output automatically filters based on active difficulties:

  • Unchecked difficulties show empty arrays []
  • Only active difficulties receive new notes
  • Switch difficulties mid-chart for varied complexity
